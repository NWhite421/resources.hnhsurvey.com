Boundary & Adjacent Information

The boundary, adjacent parcels, and roads are drafted. Each subject parcel segment has appropriate measurements (bearing and distance for lines, curve data for curves) (Reference). Easements are drafted and annotated. The subject parcel is identified with lot & block information. Adjacent lots are annotated with their lot number (and block number if the lot is in another block).

Easements

Add line work for every easement shown on the plat, either described through the typical diagram shown on the first page of the plat or as shown on the lot itself. Label each easement with its width and purpose. A typical width and description can be used if all sides are the same.

Dimensions

Add two (2) dimensions, one from each property corner, to the centerline of the right-of-way. Ensure the accuracy is showing two (2) decimal places.

Road Information

Each road directly adjacent to the parcel has: the road name (if the road name is different from the plat, make the current road name large), plat name (if applicable), and a line containing the width of the right-of-way. Draw the entire right-of-way centerline from intersection to intersection or intersection to cul-de-sac termination.

Paper Space
Legal Description

Add the legal description by either using the legal description helper (lot and block lots only) or typing out the legal description.

Boundary Survey Certified To

Fill out with the client name. If we have additional certifications, they will also go under this heading.

Property Address

The property address is added to the survey. If there is no street number available, add the parcel ID number or alternate key (ALTKEY) above the street name.

Lot Size

If the lot size is less than 21,780 square feet (0.5 acres), put the square footage of the lot. If the lot is at or above 21,780 square feet (0.5 acres), put the acreage. We do not put both elevations.

Surveyor's Note 6 - FEMA

Fill out the missing information in the FEMA note; including the map and panel number, jurisdiction, effective date, and flood zone(s). If the property lies in a "AE" zone, post the elevation.
